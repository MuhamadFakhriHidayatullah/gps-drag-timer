# GPS Drag Timer (Web)

Website ini digunakan untuk menghitung **waktu drag race motor** otomatis menggunakan **GPS** pada jarak 201 meter dan 402 meter.  
Cocok untuk latihan atau simulasi, bukan untuk lomba resmi.
